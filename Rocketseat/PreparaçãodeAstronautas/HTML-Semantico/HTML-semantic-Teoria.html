HTML5 Semântico

As tags

Elas que irão ajudar a criar um HTML semântico, pois algumas tags tem significados específicos e orientações claras sobre onde devem ficar na página e o motivo dela existir.

Exemplo:


<p>

Ser feliz sem motivo é a mais autêntica forma de felicidade.
Não sei quem

</p>


Com uma melhor semântica:

<blockquote>

Ser feliz sem motivo é a mais autêntica forma de felicidade.

<cite> - Não sei quem - </cite>

Entendendo os significados

  • Como autor da página, é a sua responsabilidade entender os significados dos elementos

  • Muitas vezes é simples, pois o HTML não possui um leque gigante de elementos

  • Algumas vezes é complexo de se entender alguns elementos.

Saiba que:

Alguns elementos, você irá usar em todos documentos que você criar, e alguns, talvez você nunca precise utilizar.
Em muitos casos, a marcação do conteúdo é uma questão de gosto pessoal e será diferente de autor para autor
Melhorar a semântica é o foco do HTML5. Sempre buscaremos regras e padrões para aprimorar a estrutura do documento.
